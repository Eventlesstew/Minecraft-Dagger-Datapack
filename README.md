# Minecraft Daggers Datapack
This datapack adds Daggers into Minecraft! This repository contains the resource pack and datapack necessary for use.

Daggers are essentially swift killers that ignore i-frames, leading to very high attack speeds. Additionally, they have reduced range and don't inflict any knockback, so it'll be a bit risky trying to land those hits in.

Damage increments are as follows:
- Wooden and Gold: 0
- Stone and Copper: 1
- Iron: 2
- Diamond: 3
- Netherite: 4

These items can be crafted in your hand by using a stick, then having the tool material on top of it. For example, a Diamond Dagger requires you to put a diamond on top of a stick. The netherite Dagger is upgraded from Diamond Daggers like normal.

As of now, no enchantments can be applied to these items.

## Technical Notes
This datapack uses the Music Disc Optimisations to allow the Diamond Dagger to have a Netherite Upgrade. 

This means all music discs obtained through survival means will use Disc 13 internally. With their item name, model and music changed to accomodate.

All daggers are "Creator" Music Discs internally. The Diamond Dagger however is the "Creator Music Box" disc so it can be upgraded to Netherite.
